Swift协议

  • 属性,方法,协议继承,聚合,关联类型,扩展
  • 聚合 protocol1 & protocol2
  • 协议检查 people is PersonProtocol
  • 协议关联类型
    protocol weightCalculableP{
    associatedtype WeightType
    var weight:WeightType{ get}
    }
  • 协议扩展: 在协议扩展中实现的方法,在其它类中可以直接拿来使用,无需实现
    • 还可以为系统类的协议扩展
      extension CustomStringConvertible{ var customDescription:String return “YQ” + description }
    • 为存在的类型采纳和遵守一个协议,无需访问现有类型的源代码
      extension Double: CustomStringConvertible { public var description:String { get } }
  • Swift标准库中为我们提供了55种协议, 基本是以 “type”,”able”,”Convertible”,分别表示该协议”可以被当作XX类型”，”具备某种能力或特性”，”能够改变或变换”

Swift中常见的协议

  • Equatable，是和比较相关的协议,需要重载 “==”
    return s1.match == s2.match && s1.english = s2.english
  • Comparable，是和比较相关的协议,遵守该协议需要重载 “<” 运算符
  • CustomStringConvertible:需要重载 description
  • ExpressibleByArrayLiteral:提供了使用数组文本初始化类型的能力,具体来说使用分割的值,实例,字面值列表,方括号创建数组文本。

    Swift中高阶函数

  • func square(a:Float) -> Float{ aa }
    func cube(a:Float) -> Float { a    a*a }
    func averageSumOfSquares(a:Float,b:Float,f:(Float -> Float)) -> Float { } // f:是一个闭包
    averageOfFunction(3, 4, square)
    averageOfFunction(3, 4, cube)
    averageOfFunction(3, 4, {(x: Float) -> Float in return x * x})
    可以通过 $index索引来捕获block中的参数列表,按照顺序 0…1
  • Swift中高阶函数
    • map函数用于将序列中的某个元素通过某种方法进行转化
      [x1,x2,x3].map(function) -> [function(x1),function(x2),function(x3)]
    • 在swift中,map被申明为Array类的命名方法。这个方法以一个命名函数transfrom为参数,transfrom负责将元素类型T转换成U并返回一个类型U的数组
      func map(transfrom:(T) -> U ) -> U[]
    • stringsArray = moneyArray.map({money in “(money)?”})
    • filter函数, filter(includeElement:(T) -> Bool) -> T[]
      fileteredArray = moneyArray.filter({$0 > 30})
    • Reduce函数
      reduce(intial:U ,combine:(U,T) -> U) -> U
      接受两个参数,一个为U类型的初始值,另一个把类型为U的元素为T的元素组合成一个类型为U的树,最总结果为一个类型为U的值

Swift中的CDD架构

  • 不书写delegate,传递事件。
  • CCD 适用于非常复杂的View层级结构。
  • subView->event-> context -> ViewController。
  • 每个subView可以获取到context,并将事件传递给Context。
  • 基于MVC的方式。CDDContext包含了⬇️
  • CDDPresenter: baseViewController,CDDView,adapter(for tableViewAdapt)
  • CDDInteractor: baseViewController
  • CDDView: CDDPresenter,CDDInteractor

  • 可以使用 cocoapods 来管理项目
  • swift中获取命名空间 namespace,动态添加类
    ClassFromString(namespace + childClassName)
  • namespace在info.plist中对应的CFBundleExecutable,
  • 也可以在BuildSettings中对 productName进行修改。
    ProductName $(TAGET_NAME) 默认为此项目,可以直接修改为nameSpace 。
